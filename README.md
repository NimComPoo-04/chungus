##### Build instructions

Required Programs:

* Java (OpenJDK 11)
* Posix complient Shell (eg. bash, dash, ksh)

Building and running the server:

```bash
./build.sh
java -jar ./build/chungus.jar
```
To connect to the server:

open your favorite webbrowser
and go the site
http://localhost:8080
